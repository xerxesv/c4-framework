Bootstrap Widgets

The C4 framework uses Bootstrap JavaScript components to handle many interactive features. In the following section, we will cover various ways that these components should be used in ebooks.

Popovers

Bootstrap's popover.js plugin is an extension of the tooltip.js plugin that allows for full aup in the tooltip.

As Key Term Tooltips

In the following example, popover.js is used to create custom tooltips. The definitions are pulled from an external #glossary document (glossary.html in this case) that houses semantic dt/dd pairs inside a definition list (dl).

Academic writing can serve a number of different purposes — to argue for what you think about a topic and why, to report on what’s known about an issue, to propose a solution for some problem, and so on.

When you write in an academic context, you’re expected to state your main point explicitly, often in a thesis statement. 

<a tabindex="0" role="button" data-placement="top" title="Arguing">
  <dfn>argue</dfn>
</a>
             
<dl id="glossary">
  <dt>Arguing</dt>
  <dd>A STRATEGY that can be used in any kind of writing to support a claim with REASONS and EVIDENCE.</dd>
</dl>
