# c4-framework

> A framework for Norton ebooks.

## To get started

```sh
$ gulp [--prod]
```

And you'll have a new Jekyll site generated for you and displayed in your
browser. Neato. If you want to run it with production settings, just add
`--prod`.

## Usage

```sh
$ gulp build [--prod]
```

```sh
$ gulp deploy
```
